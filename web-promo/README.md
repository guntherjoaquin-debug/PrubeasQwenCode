# AquaCoop ERP - Landing Page Promocional

Sitio web promocional para el sistema ERP especializado en cooperativas de agua potable.

## 📁 Estructura del Proyecto

```
web-promo/
├── index.html          # Página principal
├── styles.css          # Estilos personalizados
├── script.js           # Funcionalidad JavaScript
└── assets/             # Recursos adicionales (imágenes, etc.)
```

## 🚀 Características

- **Diseño Responsivo**: Se adapta a todos los dispositivos (móvil, tablet, desktop)
- **Tailwind CSS**: Framework de utilidad para estilos modernos
- **Font Awesome**: Iconos profesionales
- **Animaciones Suaves**: Transiciones y efectos visuales atractivos
- **Formulario de Contacto**: Funcional con validación básica
- **Navegación Intuitiva**: Menú fijo con scroll suave
- **SEO Optimizado**: Meta tags y estructura semántica

## 📋 Secciones Incluidas

1. **Inicio (Hero)**: Presentación impactante del producto
2. **Problemas/Soluciones**: Identificación de desafíos comunes
3. **Módulos**: Descripción detallada de las 9 funcionalidades principales
4. **Beneficios**: Ventajas competitivas del sistema
5. **Testimonios**: Opiniones de clientes satisfechos
6. **Contacto**: Formulario e información de contacto

## 🛠️ Cómo Usar

### Opción 1: Abrir directamente
Simplemente abre el archivo `index.html` en tu navegador web.

### Opción 2: Servidor local recomendado
Para una mejor experiencia, usa un servidor local:

```bash
# Si tienes Python instalado
python -m http.server 8000

# Luego abre en tu navegador
http://localhost:8000
```

### Opción 3: Con Node.js
```bash
# Instalar serve globalmente
npm install -g serve

# Ejecutar el servidor
serve .
```
